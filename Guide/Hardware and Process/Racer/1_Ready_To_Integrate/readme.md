# Ready To Integrate — Component Processes

Version: 1.0

This directory documents processes and instructions for individual components used by the Racer project. The layout is kept concise so technicians can find the steps and checklists quickly.

## Scope
Scope is limited to:
- Receive parts from supplier
- Inspect parts
- Configure parts (if necessary)
- (Optional) Assemble parts to form a larger component (example: Raspberry Pi + carrier board + WiFi antenna + heatsink)
