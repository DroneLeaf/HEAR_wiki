## Yakuake — Terminal emulator: installation and configuration

### Installation
- Install Yakuake via apt:
    ```bash
    sudo apt install -y yakuake
    ```

### Configuration
- Launch Yakuake from the Applications menu (do this once after each reboot).
- Press `F12` to toggle the terminal at any time.
- Open **Settings → Profile → Scrolling** and enable **Unlimited scrollback**.
